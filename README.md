### `rustc`

| Command | Description |
|---|---|
| `rustc --version` | Prints Rust version information |

### `rustup`

| Command | Description |
|---|---|
| `rustup update` | Updates Rust version |
| `rustup self uninstall` | Uninstalls Rust |
